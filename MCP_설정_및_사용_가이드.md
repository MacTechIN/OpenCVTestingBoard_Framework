# MCP 설정 및 서버 사용 가이드

이 문서는 `mcp_config.json` 파일의 오류 수정 내역과 현재 연결된 각 MCP(Model Context Protocol) 서버의 사용법을 정리한 요약본입니다.

## 1. mcp_config.json 수정 내역

설정 파일에서 발생했던 두 가지 주요 오류를 수정했습니다.

### 오류 1: notebooklm 속성 허용 안 됨
- **원인**: `notebooklm` 설정이 `mcpServers` 객체 내부에 또 다른 `mcpServers`라는 이름으로 중첩되어 있었습니다.
- **수정**: 중첩된 구조를 제거하고 `notebooklm`을 다른 서버들과 동일한 레벨로 이동시켰습니다.

### 오류 2: GitKraken 설정의 "type": "stdio" 오류
- **원인**: `GitKraken` 설정 블록에 해당 스키마에서 허용되지 않는 `"type": "stdio"` 속성이 포함되어 있었습니다.
- **수정**: 해당 속성을 제거하고, 그 뒤에 남은 불필요한 쉼표(trailing comma)를 제거하여 문법 오류를 해결했습니다.

## 2. MCP 서버별 주요 기능 및 사용법

모든 MCP 서버는 채팅창에서 자연어 명령으로 편리하게 호출할 수 있습니다.

### 🛠️ GitKraken (Git 저장소 및 협업 관리)
- **주요 기능**: 커밋, 로그 확인, PR 생성/리뷰, 이슈 추적.
- **주요 명령어**:
    - "현재 변경 사항을 커밋해줘"
    - "새로운 PR(풀 리퀘스트) 생성해줘"
    - "나에게 할당된 이슈 목록 보여줘"

### 🎨 StitchMCP (UI 디자인 및 프론트엔드 코드 생성)
- **주요 기능**: 프롬프트를 통한 UI 디자인 및 React/HTML 코드 생성.
- **주요 명령어**:
    - "모던한 관리자 대시보드 화면 만들어줘"
    - "로그인 페이지 UI 디자인하고 코드 생성해줘"

### 🧪 TestSprite (자동화 테스트 생성)
- **주요 기능**: 코드 분석을 통한 테스트 계획 수립 및 유닛/통합 테스트 자동 생성.
- **주요 명령어**:
    - "이 프로젝트의 백엔드 테스트 계획 세워줘"
    - "핵심 로직에 대한 테스트 코드 짜고 실행해줘"

### 📚 context7 (최신 기술 문서 검색)
- **주요 기능**: 라이브러리 및 프레임워크의 공식 문서와 최신 코드 예제 검색.
- **주요 명령어**:
    - "Next.js 15의 서버 컴포넌트 사용법 찾아줘"
    - "Tailwind CSS 애니메이션 예제 보여줘"

### 🎯 Dart (Flutter/Dart 개발 지원)
- **주요 기능**: 앱 실행, 핫 리로드, 코드 분석, 위젯 트리 디버깅.
- **주요 명령어**:
    - "애플리케이션 실행해줘"
    - "현재 변경 사항 핫 리로드(Hot Reload) 적용해줘"
    - "화면의 위젯 트리 구조 보여줘"

### 🔥 Firebase (백엔드 서비스 관리)
- **주요 기능**: Firebase 프로젝트 초기화, 호스팅 배포, Firestore 데이터 관리.
- **주요 명령어**:
    - "Firebase 호스팅에 배포해줘"
    - "Firestore의 'users' 컬렉션 데이터 가져와줘"

### 📓 NotebookLM (문서 기반 지식 활용)
- **주요 기능**: 업로드된 문서 내용을 바탕으로 질문 답변 및 요약 제공.
- **주요 명령어**:
    - "내 노트북에 있는 기술 문서 요약해줘"
    - "학습 가이드 문서에서 특정 개념 설명해줘"

---

## 3. 권장 사항

설정 파일(`mcp_config.json`)을 수정했으므로, 변경된 내용이 에디터에 완전히 반영되도록 **명령 팔레트(Cmd+Shift+P)에서 "Reload Window"**를 실행하시는 것을 권장합니다.
